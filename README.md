## Data Fetching scripts

To fetch POGOH data you can simply run:

```bash
python scripts/fetch_data.py --d_name pogoh --start_date 2025-05-01 --end_date 2025-06-01
```

To fetch Weather data you can run:

```bash
python scripts/fetch_data.py --d_name weather --start_date 2025-05-01 --end_date 2025-06-01 --timezone America/New_York
```
